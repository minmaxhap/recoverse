# Recoverse Design System

## Design Direction

Recoverse should feel like opening a warm letter from a past version of yourself.

Core visual keywords:

- Time capsule
- Time travel
- Paper letter
- Personal archive
- Memory box
- Soft light

Cute characters can appear as supporting details, but the user's memories are the main character.

## Character Usage

Characters should not explain the product or dominate the screen. Treat them as small archive markers that support the emotional tone.

Allowed placements:

- Empty capsule list states
- Import/export completion states
- Small supporting detail inside the rediscover card
- Brief celebration after creating the first capsule

Avoid placing characters:

- As the main visual element on the home screen
- Inside the answer writing area
- As primary button or menu icons
- Larger than the shared retrospective content itself

## Emotional Tone

- Warm
- Nostalgic
- Calm
- Slightly mysterious
- Personal

Avoid:

- Cold productivity dashboards
- Overly cute character-first styling
- Marketing landing-page composition
- Generic note-app flatness

## Key Components

### Rediscover Card

The emotional home card that resurfaces an old answer or question.

### Capsule Card

Shows capsule title, type, description, progress, and recent state.

### Question Card

Shows a question and whether it has answers.

### Answer Editor

Lets users write and save text answers one line at a time.

## Layout Principle

Design mobile-first.

```text
Home -> Capsule Detail -> Question Card / Answer Editor
```

On PC, use the wider screen to show list and detail together.
