# Recoverse Design System

## Design Direction

Recoverse should feel like opening a warm letter from a past version of yourself.

Core visual keywords:

- Time capsule
- Time travel
- Paper letter
- Personal archive
- Memory box
- Soft light

Cute characters can appear as supporting details, but the user's memories are the main character.

## Emotional Tone

- Warm
- Nostalgic
- Calm
- Slightly mysterious
- Personal

Avoid:

- Cold productivity dashboards
- Overly cute character-first styling
- Marketing landing-page composition
- Generic note-app flatness

## Key Components

### Rediscover Card

The emotional home card that resurfaces an old answer or question.

### Capsule Card

Shows capsule title, type, description, progress, and recent state.

### Question Card

Shows a question and whether it has answers.

### Answer Editor

Lets users write and save text answers one line at a time.

## Layout Principle

Design mobile-first.

```text
Home -> Capsule Detail -> Question Card / Answer Editor
```

On PC, use the wider screen to show list and detail together.
