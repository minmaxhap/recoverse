# Recoverse

> **"기억은 흘러가지 않는다 — 짧은 한 줄로 묶어두면, 시간이 지난 뒤 다른 내가 다시 만난다."**
>
> Recoverse는 SNS·다이어리 앱이 채우지 못한 자리 — 혼자 쓰고, 잊고, 다시 발견하는 회고 공간 — 을 위한 모바일 우선 회고 도구다.
> 첫 사용자는 30초 안에 한 줄을 남길 수 있고, 며칠/주/년 뒤 같은 한 줄을 다시 만난다.

Recoverse는 짧은 질문에 답하면서 자신의 기억을 남기고, 시간이 지난 뒤 다시 열어 그때의 나를 발견하는 개인 회고 앱이다.

현재 구현은 Vue 3 기반 프론트엔드 MVP이며, 데이터는 브라우저 `localStorage`에만 저장된다. 계정, 서버, 외부 분석은 없다.

## 주요 사용자

- 짧은 회고를 꾸준히 남기고 싶지만 일기 앱이 부담스러운 1인 사용자
- 1년 뒤·N년 뒤의 자신과 다시 만나는 경험을 원하는 사람
- 감정 기록을 클라우드에 올리는 게 부담스러운 프라이버시 우선 사용자

## 핵심 차별점

- **다시 만남 루프** — 작성 7일 후부터 "오늘 다시 떠오른 기억" 카드가 홈에 자동 등장 (1년/1달/1주 우선순위 + 날짜 시드 회전)
- **30초 진입** — 빈 상태에서 wizard 없이 단일 질문 한 줄 작성
- **기기 단독 저장** — 모든 회고는 이 기기 localStorage에만. 백업/공유는 사용자 의도가 있을 때만 동작

## 핵심 화면

- 홈: 오늘 다시 떠오른 기억과 첫 작성 진입
- 새 회고: 유형 → 기간 → 질문 세트를 고르는 wizard
- 회고 작성: 질문 카드 단위로 한 문장씩 답하는 흐름
- 회고 상세: 대표 문장과 질문/답변을 감상
- 다시 보기: 같은 질문 타임라인, 연도별/주제별/랜덤 다시 보기
- 함께 보기: URL 해시로 받은 읽기 전용 공유 화면
- 설정: 언어, 테마, 회고 백업/가져오기, 전체 삭제
