# Recoverse

Recoverse is a theme-based memory capsule app for remembering, reflecting, and rediscovering forgotten versions of yourself.

The current MVP is local-first and built around retrospective capsules. Yearly retrospectives are the default example, but users can create custom capsules such as travel memories, project retrospectives, career reflections, or life-stage notes.

See `recoverse-frontend/README.md` for setup, usage, backup, import, and test instructions.
