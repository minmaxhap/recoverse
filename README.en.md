# Recoverse

Recoverse is a retrospective capsule app for storing memories and reopening them later to rediscover past versions of yourself.

The current MVP is localStorage-first. Yearly retrospectives are the default example, but users can also create custom capsules for trips, projects, careers, relationships, or life stages.

## MVP Scope

- Create/delete capsules
- Create/edit/delete question cards
- Write answers
- Rediscover old answers
- Search capsules
- Compare repeated questions
- JSON export/import
- Korean/English language selection

Login, cloud storage, sharing, and PDF export are outside the MVP.
