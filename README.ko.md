# Recoverse

Recoverse는 기억을 캡슐에 담아두고, 나중에 다시 열어 과거의 나를 발견하는 회고 앱입니다.

현재 MVP는 localStorage 기반으로 동작하며, 연도 회고를 기본 예시로 제공하되 여행, 프로젝트, 커리어, 관계, 특정 시기 같은 사용자 정의 캡슐도 만들 수 있습니다.

## MVP 범위

- 캡슐 생성/삭제
- 질문 카드 생성/수정/삭제
- 답변 작성
- 다시 발견하기
- 캡슐 검색
- 반복 질문 비교
- JSON 내보내기/가져오기
- 한글/영어 언어 선택

로그인, 클라우드 저장, 공유, PDF 내보내기는 MVP 범위 밖입니다.
