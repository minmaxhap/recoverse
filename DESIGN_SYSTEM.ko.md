# Recoverse 디자인 시스템

## 디자인 방향

Recoverse는 과거의 나에게서 온 따뜻한 편지를 여는 느낌이어야 한다.

핵심 시각 키워드:

- 타임캡슐
- 시간여행
- 종이 편지
- 개인 아카이브
- 오래된 기록함
- 부드러운 빛

귀여운 캐릭터는 보조 요소로만 사용한다. 앱의 주인공은 사용자의 기억과 회고다.

## 감정 톤

- 따뜻함
- 그리움
- 차분함
- 신비로움
- 개인적인 친밀함

피할 것:

- 차가운 생산성 대시보드
- 과하게 귀여운 캐릭터 앱
- 과장된 랜딩페이지 구성
- 일반 메모앱처럼 밋밋한 느낌

## 주요 컴포넌트

### 다시 발견하기 카드

오래된 답변이나 질문을 다시 보여주는 홈의 핵심 카드다.

### 캡슐 카드

캡슐 제목, 유형, 설명, 진행률, 최근 수정 상태를 보여준다.

### 질문 카드

캡슐 안의 질문과 답변 여부를 보여준다.

### 답변 편집기

텍스트 답변을 한 줄씩 작성하고 저장한다.

## 레이아웃 원칙

모바일 우선으로 설계한다.

```text
홈 -> 캡슐 상세 -> 질문 카드 / 답변 편집
```

PC에서는 넓은 화면을 활용해 목록과 상세를 함께 보여줄 수 있다.
