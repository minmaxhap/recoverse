# Recoverse 제품 명세

## 제품 정의

Recoverse는 과거의 나를 다시 만나는 회고 캡슐 앱이다.

사용자는 연도, 특정 시기, 여행, 프로젝트, 관계, 커리어 같은 주제로 캡슐을 만들고, 캡슐 안의 질문 카드에 답변을 남긴다. 시간이 지난 뒤 캡슐을 다시 열어보며 잊고 있던 감정과 선택, 장면을 발견한다.

## 핵심 경험

> 기억을 캡슐에 담아두고, 나중에 다시 열어 과거의 나를 발견한다.

## 핵심 감정

- 따뜻함
- 그리움
- 자기이해
- 신비로움
- 조용한 발견감

## 주요 사용자

- 연말 회고를 예쁘게 정리하고 싶은 사람
- 예전 기록을 다시 보며 자신을 이해하고 싶은 사람
- 특정 시기, 여행, 프로젝트, 관계를 회고하고 싶은 사람
- 나중에 일부 회고를 친구에게 읽기 전용으로 공유하고 싶은 사람

## 핵심 구조

```text
캡슐 -> 질문 카드 -> 답변
```

## 주요 화면

### 홈

- 다시 발견하기 카드
- 내 캡슐 목록
- 새 캡슐 만들기
- 최근 수정/진행률

### 캡슐 상세

- 캡슐 제목과 설명
- 질문 카드 목록
- 답변 작성/수정
- 진행률
- 질문 추가/삭제

### 공유 방향

초기 공유는 읽기 전용만 고려한다. 로그인과 클라우드 저장은 공유 기능이 필요해질 때 도입한다.

## 성공 기준

- 앱 방향이 캡슐 중심으로 명확해진다.
- 공유 확장이 가능한 데이터 구조를 갖춘다.
- 홈에서 다시 발견하기와 내 캡슐 목록이 중심이 된다.
- UI가 타임캡슐, 종이 편지, 개인 아카이브 감성을 가진다.
